# 📝 Todo App (Flutter)

A simple and clean Todo application built using Flutter and Dart. This app helps users manage their daily tasks efficiently with an easy-to-use interface.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed (if implemented)
- 📱 Clean and responsive UI
- ⚡ Smooth performance

---

## 🛠️ Built With

- Flutter
- Dart

---

## 🎯 Purpose

This project was created to learn Flutter development, understand state management, and improve UI/UX design skills.
